# Brand Assets

Place the official logo here as `logo.png`.

The app automatically displays it in the **navbar**, **login/register pages**, and the **footer** via the `<Logo />` component (`src/components/ui/Logo.tsx`).

Until `logo.png` exists, a styled "SD" monogram placeholder is shown automatically
(the component falls back on image load error). No code changes are needed once you
add the real file — just drop `logo.png` in this folder.

Recommended: square PNG with transparent background, at least 256×256px.
