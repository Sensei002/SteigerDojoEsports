Browser Sources

Add each of these as a Browser source in OBS. Use canvas size 1920 × 1080 and tick “Refresh browser when scene becomes active”. They update live from the Match Control panel.

OBS setup tips

  • Width / Height: 1920 × 1080 for every overlay (they position themselves inside the frame).
  • Custom CSS: leave default — backgrounds are already transparent.
  • Control from another PC/phone: works only in Firebase mode. In Local mode, open the control panel as an OBS Custom Browser Dock on the same PC.
  • Right-click the source → Interact is not needed; everything is driven from the panel.
